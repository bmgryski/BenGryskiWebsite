Web Mapping Applications

Overview of Web Mapping applications integrating data from a variety of sources to solve problems in a common dashboard or interface.


US-Wildfire Response Dashboard

This dashboard aims to give an overview of all ACTIVE wildfires in the United States of America. The data is derived from a mixture of sources. The wildfire boundaries, and structure data is all ESRI data which is provided via NGA satelites that are tasked to monitor these active wildfire. The events and stability guage seen on the right hand side of the dashboard and bottom respectively are datasets provided by Geospark Analytics. This dataset utilizes open source data to determine changes in stablity trends at the local, regional, and country level.


Critical Infrastructure Dasboard

This dashboard aims to give an overview of critical infrastucture at the country level, most specifically energy. The dashboard aims to use Geospark Analytic's near real-time data to monitor critical infrastructure and threats that may arise around these assets.
